Title - Spotify Music Analytics Dashboard

Description: Spotify Music Analytics Dashboard is an interactive data visualization project that analyzes artist popularity, track trends, album distribution, genre insights, and follower statistics using Spotify dataset metrics. The dashboard provides data-driven insights into music streaming patterns and artist performance through dynamic charts and KPI indicators.

Tech Stack:
1) Power BI – Dashboard creation and interactive data visualization.
2) Spotify Dataset – Data source for artists, tracks, albums, and genres.
3) Power Query – Data cleaning and transformation.
4) DAX – KPI calculations and custom measures.
5) Microsoft Excel – Data preprocessing and storage.
6) Data Visualization Techniques – Charts, KPI cards, filters, and trend analysis visuals.

Data Source: Kaggle

Business Problem: Music streaming platforms generate massive amounts of data related to artists, tracks, albums, genres, and listener engagement. However, extracting meaningful insights from this data can be challenging without effective visualization and analysis tools.

Goals: The goal of this project is to build an interactive analytics dashboard that helps analyze:
1) Artist popularity and follower trends.
2) Track and album performance.
3) Genre distribution and music trends.
4) User engagement patterns through popularity metrics.

Project Highlights:
1) Built an interactive Spotify Music Analytics Dashboard using Power BI.
2) Analyzed 8,500+ tracks, 2,500+ artists, 4,800+ albums, and 600+ genres.
3) Visualized artist popularity, follower trends, album types, and genre distribution.
4) Implemented dynamic filters for genre and release date analysis.
5) Created KPI cards to track key music analytics metrics.
6) Used DAX measures and Power Query for data transformation and insights generation.
7) Designed a Spotify-inspired dark theme UI for better user experience.
8) Generated data-driven insights to compare artist performance and streaming trends.

Key Visuals:
1) KPI Cards: Show overall metrics such as total artists, tracks, albums, and genres for quick summary insights.
2) Donut Chart (Popularity Distribution): Compares high-popularity tracks vs total tracks to highlight how hit-driven the dataset is.
3) Bar Chart (Album Type Distribution): Displays count of Albums, Singles, and Compilations to understand release patterns.
4) Line Chart (Monthly Average Popularity): Tracks popularity trends across selected artists over time.
5) Horizontal Bar Chart (Long Songs by Artist): Shows which artists have more long-duration tracks.
6) Bar Chart (Followers by Artist): Compares total follower count across top artists to measure audience reach and influence.
7) Filters: Enable dynamic exploration by genre and release date for deeper analysis.

Key Insights:
1) Taylor Swift has the highest follower count among all artists in the dataset, indicating strong global audience engagement.
2) Most music releases are categorized as Albums, while Compilations have the lowest share.
3) Only a smaller percentage of tracks achieve high popularity, showing that a few songs dominate streaming attention.
4) Artists like HUNTR/X and Saja Boys maintain high average monthly popularity scores.
5) The dataset contains a wide variety of music with 662 genres and over 8,500 tracks, highlighting strong genre diversity.
6) Long-duration songs are more commonly associated with globally popular artists such as Taylor Swift, The Weeknd, and Lana Del Rey.
7) Popularity trends reveal differences in listener engagement across artists and release types.
